## 缺陷严重程度说明

| 级别 | 说明 | 示例 |
|------|------|------|
| 🔴 致命 | 导致应用崩溃或核心功能完全不可用 | 输入法无法启动、拼音输入完全失效 |
| 🟠 严重 | 核心功能部分不可用或数据丢失 | 候选字不显示、收藏数据丢失 |
| 🟡 一般 | 非核心功能异常或体验不佳 | 主题切换不生效、按键动画卡顿 |
| 🟢 轻微 | 显示问题或文案错误 | 按键标签对齐偏差、翻译错误 |
