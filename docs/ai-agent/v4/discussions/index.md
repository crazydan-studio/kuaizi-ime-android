# 讨论记录索引

本目录存放与用户讨论的记录文档。文档以三位数字开头和讨论内容命名。文档为历史记录，不对已有内容做更新。

讨论文档主要记录答疑解惑、方案优劣探讨等具备知识沉淀的、包含思维过程的内容，不包含决策、指令、要求等命令性内容，也不包含执行了什么任务等内容。

