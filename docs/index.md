## 项目概述

筷字输入法是一款 Android 平台上的开源中文拼音滑行输入法，核心设计理念：

- **无联网**：不采集、不上传任何用户数据，完全离线运行
- **无跟踪**：不含任何统计、分析或追踪组件
- **隐私优先**：所有用户数据（输入历史、收藏等）仅存储在本地设备

项目采用 MVI 架构，基于 Kotlin 协程 + Flow 和 Jetpack Compose，通过 ImeIntent + StateFlow 实现单向数据流。
