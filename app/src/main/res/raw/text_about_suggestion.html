本应用通过 腾讯兔小巢 以匿名方式收集反馈信息。 为了便于查看回复内容，系统仅会在每个安装应用首次访问时为其创建一个随机帐号， 该帐号仅包含一个随机的字符串作为其唯一标识，一个随机的用户名称（形式为 筷友123456） 以及一个通过 Multiavatar API  生成的随机头像（头像不用于区分地域、国别、宗教、肤色、种族、年龄、性别等身份信息）， 并且该帐号也不会与任何设备、任何个人建立对应关系， 除非在首次提交反馈时开通了微信提醒。


此外，为方便进行问题排查，在每次提交的反馈中还会附加以下系统和应用信息（仅对本应用反馈服务的后台管理员可见）：

  • 系统信息：%1$s，含设备生产商名称、设备型号、操作系统版本号、API 版本号、屏幕分辨率；
  • 应用信息：%2$s，含本应用包名、本应用版本号；
